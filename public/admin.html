📅 Importer le calendrier

📋
ℹ️ Le fichier doit contenir les colonnes : NoPartie, Date, Salle, NomAnimateur, NoQuestionnaire, NoÉquipeA, NoÉquipeB, NoÉquipeQuestionnaire, LienReunion.
Les colonnes LienConsole et LienBouton sont ignorées. Le fichier remplace entièrement le calendrier existant.

📥 Importer un questionnaire

📊
ℹ️ Le fichier doit contenir une feuille Vers BD avec les tables tblThèmes et tblQuestionnaires.
Le numéro de questionnaire doit correspondre à celui dans parties.json.
